# ✈️ Airline Data Warehouse & BI Project

## 📌 Overview
This project demonstrates a complete Data Warehouse and Business Intelligence solution built using airline on-time performance data from the U.S. Department of Transportation (BTS).

## 🏗️ Architecture
- Star Schema Data Warehouse (Fact & Dimension tables)
- ETL pipeline using SSIS
- SQL Server stored procedures for analytics
- SSRS reporting
- Tableau dashboard for visualization
- Neo4j graph database for relationship analysis

## 🔍 Key Insights
- Top 10 worst performing airports identified
- Carrier delay performance comparison
- Monthly trend analysis of flights vs delays
- Delay causes analysis (weather, NAS, carrier, security)

## ⚙️ Tech Stack
SQL Server | SSIS | SSRS | Tableau | Neo4j | T-SQL

## 📁 Project Structure
- SQL Scripts: Stored procedures and schema
- ETL: SSIS pipeline logic
- Reports: SSRS outputs
- Dashboard: Tableau visualization
- Neo4j: Graph queries and relationships
